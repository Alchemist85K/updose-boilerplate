# updose-boilerplate/supabase-js

Supabase rules and best practices for AI coding agents.

## Install

```bash
npx updose add Alchemist85k/updose-boilerplate/supabase
```

## What's Included

### Rules (always active)

| File | Description |
|---|---|
| `supabase.md` | Security rules: service_role key protection, client types, RLS guidance |

### External Skills (via skills.json)

| Skill | Description |
|---|---|
| [`supabase-postgres-best-practices`](https://skills.sh/supabase/agent-skills/supabase-postgres-best-practices) | Postgres query optimization, indexing, schema design, RLS performance |
